# M47 — Product Clarity & Signature Loop

## Teswa signature loop (launch framing)
1. Discover value around you.
2. Show what you have (and what has a story).
3. Open a human conversation.
4. Move toward a real swap.

This loop is now expressed as behavior across existing surfaces, not as a new feature system.

## Surface jobs (clear separation)
- **Home — "Your Teswa now"**
  - Personal entry point for what matters *now* (living world recap, next best action, fresh stories/items).
  - Emphasizes momentum: items and stories are meant to move toward exchange.
- **Discover — "Find value intentionally"**
  - Intent-driven exploration surface for targeted discovery.
  - Uses signals (intelligence, story highlights, visual rails, filters) to help users find opportunities worth engaging.
- **Motion — "See Teswa moving"**
  - Immersive live pulse of stories, local activity, and item motion.
  - Video-led and energy-led; frames Teswa as a living exchange world.

## Differentiators surfaced more clearly (existing capabilities)
- **Living World recap**: reframed as the user's real-time Teswa world and next step, not an abstract status card.
- **Story → conversation continuity**: copy now clarifies that stories can start meaningful conversations that lead to exchange.
- **City Pulse / local activity**: Motion framing reinforces locality and nearby movement as a core signal.
- **Item-to-swap progression**: Home + Discover copy emphasizes that items are not passive listings; they can move into offers/swaps.

## What changed in M47
- Sharpened onboarding copy to explain Teswa in the first minute as discovery + showing value + human connection + swap path.
- Updated Home hero and section language to frame Home as daily command center.
- Added a compact "How Teswa moves" reinforcement strip on Home (lightweight, non-tutorial).
- Refined Discover hero/path framing to emphasize intentional discovery and decision support.
- Refined Discover story highlights copy to connect story context with starting conversation.
- Refined Motion hero, CTA, pulse intro, and empty-state CTA to clarify Motion as Teswa's visual heartbeat.

## What was intentionally NOT changed
- No new major features, systems, or route expansions.
- No compliance/safety/legal workflow changes.
- No notification backend/taxonomy changes from M46A/M46B.
- No large UI redesign; only copy/hierarchy/positioning improvements to existing surfaces.
